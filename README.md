# Remembrall

Aplicativo de lista de compras desenvolvido em Flutter como projeto acadêmico para a disciplina de Dispositivos Móveis.

A proposta do app é ajudar o usuário a organizar itens de compra de forma simples e prática, evitando esquecimentos durante idas ao mercado.

## Sobre o projeto

O **Remembrall** é um app mobile criado para cadastrar e consultar listas de compras.

O projeto foi desenvolvido com foco em praticar conceitos de desenvolvimento mobile utilizando Flutter e Dart, incluindo criação de telas, navegação, organização de código e armazenamento local de informações.

## Objetivo

O objetivo principal do projeto é oferecer uma aplicação simples para gerenciamento de listas de compras, permitindo que o usuário registre os itens que precisa comprar e consulte essas informações quando necessário.

Além da funcionalidade principal, o projeto também serviu como prática para desenvolvimento de aplicações multiplataforma com Flutter.

## Funcionalidades

- Cadastro de itens para lista de compras
- Visualização dos itens cadastrados
- Organização de informações em telas
- Navegação entre páginas do aplicativo
- Estruturação do projeto em Flutter
- Armazenamento local de dados

## Tecnologias utilizadas

- Flutter
- Dart
- SQLite / Sqflite
- Path Provider
- Material Design

## Como executar

Para executar este projeto localmente, é necessário ter o Flutter instalado e configurado na máquina.

Clone o repositório:

```bash
git clone https://github.com/carolinakoike/remembrall.git
```

Acesse a pasta do projeto:

```bash
cd remembrall
```

Instale as dependências:

```bash
flutter pub get
```

Execute o app:

```bash
flutter run
```

## Verificação do ambiente

Caso tenha problemas ao executar o projeto, verifique a instalação do Flutter com:

```bash
flutter doctor
```
